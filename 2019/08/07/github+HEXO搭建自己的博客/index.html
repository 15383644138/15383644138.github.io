一、安装node.js、Hexo和一些环境

安装node.js和hexo框架

首先需要安装nodejs，nodejs的下载网址为https://nodejs.org/en/，如下图所示：

安装完成后，打开node.js命令行，打开终端


我们可以查看下载的版本

查看node版本
node -v

查看npm版本
npm -v

另外还需要安装淘宝的cnpm包管理工具，通过下面的命令来进行安装，原因是自带的源下载特别慢

npm install -g cnpm --registry=https://registry.npm.taobao.org

输入cnpm,查看cnpm是否安装成功
cnpm

出现帮助代码则说明安装成功

然后通过cnpm来安装hexo博客框架：
cnpm install -g hexo-cli

这样hexo博客框架就安装好了，非常的简单，我们来查看下它的版本：

hexo -v

安装git

从官网下载git
一直点下一步，直到完成

二、博客搭建

本地搭建

现在使用hexo来正式搭建我们的博客。首先进入到自己指定的文件夹下面，创建一个”blog“文件夹，名字自己想怎么起就怎么起。

之后关于博客的所有内容都会在该文件夹下生成，如果在后续的搭建博客的过程中出现了难以解决的问题，直接删掉这个文件夹，重新再来即可。

进入目录，右键选择“git bash”

在所创建的目录中，初始化创建一个博客：
hexo init

在初始化的过程中，时间有点长，可能会由于网络原因而初始化失败，如果失败了，删掉创建的文件夹，再创建，重新初始化，多尝试，主要还是得解决好网络问题。

成功后，返回下列信息：

blog目录如下：

继续执行以下指令 （安装依赖包）
npm install

构建，生成静态文件这时候会有public文件夹了
hexo generate

启动服务，关闭服务是ctrl+c
hexo server

或者

hexo s(简写)


接下来，在浏览器中输入上面的，即可打开博客页面。
http://localhost:4000/

以上的搭建只能在我们的本地访问，但是在实际开发中，我们需要他人也可以远程访问我们的博客，接下来，将我们的博客部署到github上来公开使用。

注：

  • hexo相关命令均在站点目录下，用Git Bash运行。

  • 站点配置文件：站点目录下的_config.yml。

  • 主题配置文件：站点目录下的themes文件夹下的，主题文件夹下的_config.yml。

  • 启动服务器。在路径下，命令行（即Git Bash）输入以下命令，运行即可：

hexo server

  • 浏览器访问网址： http://localhost:4000/

部署到github

注册github

新建一个github仓库，命名规则为你的用户名.github.io（必须是你的用户名，其它名称无效），比如说，我的github用户名是wayxz，那么你就新建wayxz.github.io的仓库，将来我的网站访问地址就是 https://wayxz.github.io 。


在blog目录下打开git bash ，配置用户名和邮箱

输入
git config --global user.name "你的GitHub用户名"

git config --global user.email "你的GitHub注册邮箱"

生成ssh密钥文件

输入ssh-keygen -t rsa -C "你的GitHub注册邮箱"

进去复制全部

在github的设置里找到

在Git Bash中检测GitHub公钥设置是否成功，输入 ssh git@github.com

输入yes
显示successful 如下则成功
